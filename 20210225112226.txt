<script>
jQuery(document).ready(function(){

var d = document;

// 著者 META と著者 DOM のチェック
if(!d.getElementById('byline') || !d.getElementById('TMP_JointWriterID')) return false;

// 著者 META の取得
var authorID = d.getElementById('TMP_JointWriterID').content;

// 著者 META content のチェック
if(authorID == '') return false;

// 著者 META を配列にする
var authorIDs = authorID.split(',');

// 著者 DOM の取得
var authorDOM = d.getElementById('byline');

// 著者 HTML の取得
var authorHTML = authorDOM.innerHTML;

// 著者 HTML の分解
// [0] = <b>[</b>著者名1, 著者名2，<b>シグネチャ</b>
// [1] = <b>[</b>
// [2] = 著者名1, 著者名2
// [3] = ，<b>シグネチャ</b>
var authorHTMLs = authorHTML.match(/^(<b>\[<\/b>)(.+)(，<b>.+<\/b>)$/);

// 正しく分解できているかチェック
if(authorHTMLs == null) return false;
if(authorHTMLs.length != 4) return false;

// 著者名を配列にする
var authorTXTs = authorHTMLs[2].split(', ');

// 著者名を著者インデックスへのリンク付きに変換
var rep_authorTXTs = [];
for(var i = 0; i < authorTXTs.length; i++){
	rep_authorTXTs.push('<a href="//www.itmedia.co.jp/author/' + authorIDs[i] + '/">' + authorTXTs[i] + '</a>');
}

// リンク付き著者名をオリジナルと入れ替える
authorHTMLs[2] = rep_authorTXTs.join(', ');

// 著者 DOM 内に HTML を戻す

<p><A HREF="https://www.atmarkit.co.jp/ait/subtop/features/dotnet/dotnettips_index.html">連載目次</A></p>
<div class="box cmsBox cmsBox cmsBoxGreen" id="green">
<p>本稿は2004/08/27に初版公開、2004/09/02に改訂された記事を再改訂し、C# 6.0／Visual Basic 14（以下、VB 14）の補完文字列についての記述を追加したものです。また、本稿のコードは全てC# 6.0／VB 14で動作を確認しました。</p>
<br clear="all"><span></span></div>
<p>　.NET Frameworkのクラスライブラリでは、日付や時刻の情報は、DateTime構造体（System名前空間）を使って扱うことができる。本稿では、このDateTimeオブジェクトを指定した形式の文字列に変換するための基本的な方法について示す。</p>
<p>　また、C# 6.0／VB 14では、「補間文字列」（interpolated string）という機能を使うことで、同じ内容のコードをより簡潔に記述することも可能だ。補間文字列を使用したコードを知りたい方は<A HREF="//www.atmarkit.co.jp/fdotnet/dotnettips/200customdate/#interpolateddate" target="_blank">こちら</A>を参照してほしい。</p>
<h4>標準書式指定とカスタム書式指定</h4>
<p>　DateTime構造体には、日付や時刻の情報を文字列に変換するために、大きく次の三つの方法がある。</p>
<p><span class="cmsColorRed"><strong>（1）</strong></span> ToLongDateString、ToLongTimeString、ToShortDateString、ToShortTimeStringなどのメソッドを呼び出す</p>
<p><span class="cmsColorRed"><strong>（2）</strong></span> 標準書式指定文字列（d、D、t、T、f、F、g、Gなど）をパラメーターに指定して、ToStringメソッドを呼び出す</p>
<p><span class="cmsColorRed"><strong>（3）</strong></span> カスタム書式指定文字列（後述）をパラメーターに指定して、ToStringメソッドを呼び出す</p>
<p>　<span class="cmsColorRed"><strong>（1）</strong></span>に挙げたメソッドは、<span class="cmsColorRed"><strong>（2）</strong></span>の方法によるToString("D")、ToString("T")、ToString("d")、ToString("t")などの呼び出しを置き換えたにすぎない（標準書式指定文字列については、MSDNの「<A HREF="https://msdn.microsoft.com/ja-jp/library/az4se3k1(v=vs.110).aspx" target="_blank">標準の日時書式指定文字列</A>」を参照）。いずれにせよ、<span class="cmsColorRed"><strong>（1）</strong></span>と<span class="cmsColorRed"><strong>（2）</strong></span>の方法はカルチャ情報に依存した文字列が出力される。つまり、例えば、以下のコードを実行すると、その結果が日本語環境では「2004年8月24日 20:23:06」などになるが、英語環境では、これは「Tuesday, August 24, 2004 8:23:06 PM」といった値となる。</p>
<div class="box cmsBoxNoborder cmsBox">
<div class="cmsBodyBox cmsBodyBoxBlue">
<div class="cmsBodyBoxInner">
<p>DateTime.Now.ToString(<span class="cmsColorRed">&quot;F&quot;</span>)</p>
</div>
<div class="cmsBodyBoxCaption"><strong>現在の時刻を完全な日付と時刻のパターン（長い形式の時刻）に変換するコード</strong></div>
</div>
<br clear="all"><span></span></div>
<p>　カルチャ情報に依存した文字列表記は、単純に日時を画面に表示したい場合などには便利であるが、日時を文字列に変換してデータ項目の一つとして利用したいような場合（例えば、ファイル名として使用するために「20040824202306」という文字列を得たい場合）などには使えない。このような場合には、<span class="cmsColorRed"><strong>（3）</strong></span>のカスタム書式指定文字列による文字列変換を利用すると、その出力形式を自由に指定できる。</p>
<h4>カスタム書式指定文字列による変換</h4>
<p>　カスタム書式指定文字列では、書式指定子の組み合わせにより、独自の出力形式でDateTimeオブジェクトを文字列に変換できる。例えば次の例では、その結果が「20040824202306」などとなる。</p>
<div class="box cmsBoxNoborder cmsBox">
<div class="cmsBodyBox cmsBodyBoxBlue">
<div class="cmsBodyBoxInner">
<p>DateTime.Now.ToString(<span class="cmsColorRed">&quot;yyyyMMddHHmmss&quot;</span>)</p>
</div>

</div>

<h4>C# 6.0／VB 14で追加された補間文字列機能を使用する</h4>
<a name="interpolateddate"></a>
<p>　C# 6.0／VB 14では「<A HREF="https://msdn.microsoft.com/ja-jp/library/dn961160.aspx" target="_blank">補間文字列</A>」を使用することで、より簡潔な記述で、DateTime構造体を文字列に変換できる。</p>
<p>　補間文字列は先の例の後半で使用している複合書式指定における書式指定項目のインデックス（「{0}」「{1}」における「0」「1」など）の部分に直接、変数や式などを記述するものだと考えると分かりやすい。また、文字列の先頭には「$」記号を付加する。</p>
<div class="box cmsBoxNoborder cmsBox">
<div class="cmsBodyBox cmsBodyBoxBlue">
<div class="cmsBodyBoxInner">
<p><span class="cmsColorGreen"><strong>$</strong></span><span class="cmsColorRed">&quot;字面通りに表示されるテキスト <strong>{ 補間式, 文字列の幅:書式指定子}</strong> ……&quot;</span></p>
</div>
<div class="cmsBodyBoxCaption"><strong>補間文字列</strong><br>「字面通りに表示されるテキスト」「文字列の幅」「書式指定子」は省略可能。補間式に指定した要素が評価され、その結果が文字列に埋め込まれる。</div>
</div>
<br clear="all"><span></span></div>
<p>　「,文字列の幅」には文字列の幅を、「:書式指定子」には上で見た標準書式指定文字列あるいはカスタム書式指定文字列を指定できる（「文字列の幅」「書式指定子」のいずれも省略可能）。</p>
<p>　例えば標準書式指定文字列の「F」（完全な日付と時刻のパターン。長い形式の時刻）を使用して、現在の時刻をコンソールに表示するには次のようにする。</p>
<div class="box cmsBoxNoborder cmsBox">
<div class="cmsBodyBox cmsBodyBoxBlue">
<div class="cmsBodyBoxInner">
<p><span class="cmsColorOrange">var</span> now = DateTime.Now;<br>Console.WriteLine(<span class="cmsColorGreen"><strong>$</strong></span><span class="cmsColorRed"><strong>&quot;{now:F}&quot;</strong></span>);</p>
</div>
</div>
<div class="cmsBodyBox cmsBodyBoxGreen">
<div class="cmsBodyBoxInner">
<p><span class="cmsColorBlue">Dim</span> now = DateTime.Now<br>Console.WriteLine(<span class="cmsColorGreen"><strong>$</strong></span><span class="cmsColorRed"><strong>&quot;{now:F}&quot;</strong></span>)</p>
</div>
<div class="cmsBodyBoxCaption"><strong>標準書式指定文字列を使用したサンプル（上：C#、下：VB）</strong></div>
</div>
<br clear="all"><span></span></div>
<p>　カスタム書式指定文字列を指定した場合の例は次のようになる。</p>
<div class="box cmsBoxNoborder cmsBox">
<div class="cmsBodyBox cmsBodyBoxBlue">
<div class="cmsBodyBoxInner">
<p><span class="cmsColorOrange">var</span> now = DateTime.Now;<br>Console.WriteLine(<span class="cmsColorGreen"><strong>$</strong></span><span class="cmsColorRed"><strong>&quot;{now:yyyyMMddHHmmss}&quot;</strong></span>);</p>
</div>
</div>
<div class="cmsBodyBox cmsBodyBoxGreen">
<div class="cmsBodyBoxInner">
<p><span class="cmsColorBlue">Dim</span> now = DateTime.Now<br>Console.WriteLine(<span class="cmsColorGreen"><strong>$</strong></span><span class="cmsColorRed"><strong>&quot;{now:yyyyMMddHHmmss}&quot;</strong></span>)</p>
</div>
<div class="cmsBodyBoxCaption"><strong>標準書式指定文字列を使用したサンプル（上：C#、下：VB）</strong><br>これは「カスタム書式指定文字列による変換」の冒頭で見た書式指定子をそのまま使っている。</div>
</div>
<br clear="all"><span></span></div>
<p>　よって、上のサンプルプログラムは次のように書き換えられる（なお、補間文字列を使用した場合、ToStringメソッドは不要になるので、上のサンプルプログラムと全く同じわけではないことに注意。書式指定文字列が異なるものだけをピックアップしてある）。</p>
<div class="box cmsBoxNoborder cmsBox">
